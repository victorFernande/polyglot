# 🇷🇺 Imersão — Música Russa e Textos Curtos

## Materiais iniciais
- Músicas lentas com letra em cirílico.
- Posts/tweets curtos para leitura de reconhecimento.
- Vídeos A1 com fala lenta.

## Como usar
1. Identifique letras conhecidas.
2. Circule palavras repetidas.
3. Leia em voz alta com áudio.
4. Traduza apenas a ideia geral, não palavra por palavra.

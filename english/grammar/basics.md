# 🇬🇧 Inglês — Gramática de Referência

## Uso nesta trilha
Esta pasta não é para aprender inglês do zero, mas para usar o inglês como referência comparativa ao estudar outras línguas.

## Pontos úteis para comparação
- Ordem SVO: `I drink coffee`.
- Auxiliares: `do`, `does`, `did`, `will`, `would`.
- Tempos verbais como referência para alemão/francês/russo/japonês.
- Phrasal verbs e idioms.

## Frases modelo
- I would like a coffee.
- Where is the station?
- My name is Victor.
- I am learning German/French/Russian/Japanese.

# 🇬🇧 Inglês — Imersão

## Materiais úteis
- Podcasts e vídeos técnicos.
- Séries/filmes com fala natural.
- Artigos de tecnologia, ciência e cultura.
- Letras de música para expressão idiomática.

## Como usar como ponte
1. Escolha um tema em inglês.
2. Explique o mesmo tema em uma das línguas-alvo.
3. Compare vocabulário, ordem das palavras e expressões.

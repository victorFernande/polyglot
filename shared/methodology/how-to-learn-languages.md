# 🧠 Metodologia de Aquisição de Línguas

> Base científica para o projeto Polyglot.

---

## Princípios Fundamentais

### 1. Input Compreensível (Stephen Krashen)

**A regra:** Aprendemos línguas quando entendemos mensagens, não quando memorizamos regras.

**Como aplicar:**
- 70% do tempo: ouvir/ler material que você **quase** entende
- Use legendas bilingues (alemão + português) → depois só alemão → depois nenhuma
- Não se preocupe com palavras desconhecidas — entenda o contexto

**No Polyglot:**
- Rammstein com letra + tradução → letra só → sem letra
- Podcasts alemães para iniciantes (Deutsche Welle)
- YouTube com legendas alemãs

### 2. Spaced Repetition System (SRS)

**A regra:** Revisamos no momento ótimo — antes de esquecer.

**Como aplicar:**
- Anki: flashcards com algoritmo SM-2
- 10-15 min/dia, todos os dias
- Inclua áudio + imagem + contexto

**Estrutura do card ideal:**
```
Frente: Áudio da palavra + contexto (frase de Rammstein)
Verso: Significado + fonética + imagem mental
```

### 3. Active Recall vs. Passive Review

**Passivo (fraco):** Ler a lista de palavras  
**Ativo (forte):** Tentar lembrar a palavra antes de virar o card

**Como aplicar:**
- Sempre tente responder antes de ver a resposta
- Escreva frases com palavras novas
- Fale sozinho descrevendo seu dia

### 4. Shadowing (Alexander Argüelles)

**A regra:** Repetir em voz alta imediatamente após o nativo.

**Como aplicar:**
- Escute uma frase → repita imediatamente → escute outra → repita
- Comece com material lento (0.8x)
- Aumente a velocidade gradualmente

**No Polyglot:**
- Shadowing com letras de Rammstein
- Repetir frases de podcasts
- Imitar entonação de Till Lindemann

### 5. Comprehensible Output (Merrill Swain)

**A regra:** Produzir a língua (falar/escrever) força o cérebro a organizar o conhecimento.

**Como aplicar:**
- 10% do tempo: falar sozinho, escrever diário
- Não precisa estar perfeito — precisa estar comunicando
- Gravar-se e ouvir depois

### 6. Affective Filter (Krashen)

**A regra:** Ansiedade bloqueia aquisição. Relaxar = aprender mais.

**Como aplicar:**
- Use material que você **gosta** (Rammstein!)
- Não se compare com nativos
- Celebre pequenas vitórias
- Erros são dados, não falhas

## 📊 Distribuição de Tempo Ideal

```
70% Input (ouvir/ler)
  ├── 40% Input ativo (foco total, anotando)
  └── 30% Input passivo (música, podcast enquanto faz outra coisa)

20% SRS (Anki/revisão)

10% Produção (falar/escrever/shadowing)
```

**Sessão diária de 45 min:**
- 20 min: Input ativo (letra de Rammstein + tradução)
- 10 min: SRS (Anki)
- 10 min: Shadowing
- 5 min: Produção (falar sozinho/escrever)

## 🧬 Por Que Funciona

### O Cérebro e Línguas

1. **Neuroplasticidade:** O cérebro cria conexões quando exposto a padrões repetidos
2. **Chunking:** Palavras frequentemente juntas viram "blocos" (ex: "Du hast mich")
3. **Pattern Recognition:** Gramática é aprendida implicitamente, não explicitamente
4. **Emotional Memory:** Música ativa a amígdala → memória mais forte

### Por Que Rammstein Funciona

| Fator | Benefício |
|-------|-----------|
| Repetição | Reforço natural |
| Ritmo | Padrões prosódicos memorizáveis |
| Emoção | Amígdala ativada = memória duradoura |
| Repetição de frases | Chunking natural |
| Vocabulário variado | Exposição a múltiplos registros |
| Pronúncia teatral | Articulação clara para imitar |

## 🚫 O Que NÃO Funciona

| Método | Por Que Falha |
|--------|---------------|
| Memorizar listas de palavras | Sem contexto = sem memória |
| Estudar gramática antes de input | Regras abstratas sem aplicação |
| Traduzir mentalmente palavra por palavra | Lento, quebra fluência |
| Focar em erros | Aumenta filtro afetivo |
| Estudar 2h uma vez por semana | Sem repetição espaçada = esquecimento |
| Comparar com nativos | Desmotivador, filtro afetivo alto |

## 🎯 Expectativas Realistas

| Tempo | O Que Esperar |
|-------|---------------|
| 1 semana | Ler o alfabeto, reconhecer sons básicos |
| 2 semanas | Ler palavras simples em voz alta |
| 1 mês | Entender frases simples em contexto |
| 2 meses | Seguir diálogos básicos com legendas |
| 3 meses | Conversação básica (apresentação, pedidos) |
| 6 meses | Entender material autêntico simples |
| 1 ano | Conversação intermediária, leitura básica |

**Lembre-se:** Cada pessoa é diferente. Consistência > intensidade.

# 🇫🇷 Imersão — Chanson Française

## Materiais iniciais
- Músicas lentas e claras para reconhecimento de sons.
- Cenas curtas de filmes com legenda em francês.
- Podcasts A1 ou francês lento.

## Como usar
1. Ouça uma vez sem legenda.
2. Ouça com legenda/transcrição.
3. Anote 5 palavras úteis.
4. Faça shadowing de 2–3 frases.

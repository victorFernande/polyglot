# 🇯🇵 Imersão — Anime/Manga

## Materiais iniciais
- Cenas curtas com fala clara.
- Mangás simples ou diálogos com furigana.
- NHK Easy para leitura graduada.

## Como usar
1. Assista uma cena curta sem pausar.
2. Reassista com legenda japonesa/portuguesa.
3. Anote partículas e frases repetidas.
4. Faça shadowing de 1–2 falas.
